CUPPON Page

Este sitio de cupones fue desarrollado como parte del proceso formativo en la carrera de desarrollo Front-End + React en Desafío Latam. En este proyecto se aplicaron conocimientos previamente adquiridos en maquetación con HTML y CSS, integrando el framework Bootstrap 5 para estructurar y estilizar de manera ágil y responsiva.

El diseño del sitio se construyó utilizando el sistema de cards de Bootstrap para presentar cada cupón de forma ordenada, clara y visualmente atractiva. Se trabajó con grillas, espaciados, tipografías y componentes del framework para lograr un diseño adaptable a distintos dispositivos.

Además, se reutilizó el aprendizaje previo integrando íconos mediante Font Awesome, mejorando la comunicación visual y la experiencia de usuario. Este proyecto consolidó habilidades clave en el uso de frameworks CSS, diseño mobile-first y organización eficiente del contenido web.
